Powerful Features

LazyLua extends Lua with essential functionality for complex applications

High-Precision Math

Perform addition, subtraction, multiplication, and division of arbitrarily large non-negative integers without precision loss.

Enhanced Strings

Handle UTF-8 encoded strings with proper character access, length calculation, substring operations, splitting, and replacement.

Sorting Algorithms

Implement index-based bubble sort and quicksort with support for both ascending and descending order arrangements.

Get Started

Install LazyLua in your project with just a few simple steps

Installation

Download the LazyLua.lua file into your project directory, then include it in your Lua script:

-- If the file is in the same directory
dofile("LazyLua.lua")

-- Or if package.path is configured
require("LazyLua")

Initialize the modules you need:

-- Initialize math module
Mathinit()

-- Initialize string module
Stringinit()

-- Initialize sorting module
Sortinit()

Usage Examples

See LazyLua in action with these practical examples

Mathematical Operations 

Mathinit() -- Initialize the math module

-- High-precision addition
local result = LazyLua.Math.add(
    "12345678901234567890", 
    "98765432109876543210"
)
print(result) -- Output: 111111111011111111100

-- High-precision multiplication
local product = LazyLua.Math.mul(
    "123456789", 
    "987654321"
)
print(product) -- Output: 121932631112635269

String Processing 

Stringinit() -- Initialize the string module

local str = "Hello, 世界! 🌍"
print(LazyLua.String.len(str)) 
-- Output character count: 9

print(LazyLua.String.at(str, 8)) 
-- Output the 8th character: 界

print(LazyLua.String.substr(str, 1, 5)) 
-- Output substring: Hello

Sorting 

Sortinit() -- Initialize the sorting module

local numbers = {"5", "2", "8", "1"}
local sorted_indices = LazyLua.Sort.quick(
    numbers, 
    LazyLua.Sort.ASC
)
print(LazyLua.Sort.applyIndices(
    sorted_indices, 
    numbers
)) 
-- Output sorted array: 1, 2, 5, 8

API Reference

Comprehensive documentation of LazyLua's functionality

LazyLua.Math

  • add(a, b) High-precision addition
  • sub(a, b) High-precision subtraction
  • mul(a, b) High-precision multiplication
  • div(a, b) High-precision division (integer part)
  • greaterThan(a, b) Check if a > b
  • lessThan(a, b) Check if a < b
  • normalize(s) Normalize a number string

LazyLua.String

  • len(str) Get character length
  • at(str, index) Get character at index
  • substr(s, start, end) Get substring
  • split(str, delimiter) Split string
  • replace(s, old, new, limit) Replace substrings
  • includes(s, substr) Check if contains substring

LazyLua.Sort

  • bubble(arr, method) Bubble sort (returns indices)
  • quick(arr, method) Quick sort (returns indices)
  • applyIndices(i, arr) Apply indices to reorder array
  • Sort.ASC Ascending order constant
  • Sort.DESC Descending order constant
