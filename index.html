Powerful Features

LazyLua extends Lua with essential functionality for complex applications

High-Precision Math

Perform addition, subtraction, multiplication, and division of arbitrarily large integers (including negative numbers) without precision loss. Optimized for performance.

Enhanced Strings

Handle UTF-8 encoded strings with proper character access, length calculation, substring operations, splitting, joining, and advanced search capabilities.

Sorting Algorithms

Implement index-based bubble sort and quicksort with support for both ascending and descending order arrangements. Stable and fast options available.

Get Started

Install LazyLua in your project with just a few simple steps

Installation

Download the LazyLua.lua file into your project directory, then include it in your Lua script:

-- If the file is in the same directory
dofile("LazyLua.lua")

-- Or if package.path is configured
require("LazyLua")

Initialize the modules you need:

-- Initialize math module
Mathinit()

-- Initialize string module
Stringinit()

-- Initialize sorting module
Sortinit()

Usage Examples

See LazyLua in action with these practical examples

Mathematical Operations 

Mathinit() -- Initialize the math module

-- High-precision addition
local result = LazyLua.Math.add(
    "12345678901234567890", 
    "98765432109876543210"
)
print(result) -- Output: 111111111011111111100

-- Negative number operations
local result2 = LazyLua.Math.add("-123", "456")
print(result2) -- Output: 333

-- Division with precision control
local div_result = LazyLua.Math.div(
    "100000000000000000000", 
    "3", 
    10 -- 10 decimal places
)
print(div_result) -- Output: 33333333333333333333

-- Large number multiplication
local big_result = LazyLua.Math.mul(
    "123456789012345678901234567890", 
    "987654321098765432109876543210"
)
print(big_result)

Sorting 

Sortinit() -- Initialize the sorting module

local numbers = {"5", "2", "8", "1"}
local sorted_indices = LazyLua.Sort.quick(
    numbers, 
    LazyLua.Sort.ASC
)
print(LazyLua.Sort.applyIndices(
    sorted_indices, 
    numbers
)) 
-- Output sorted array: 1, 2, 5, 8

-- For descending order
local desc_indices = LazyLua.Sort.quick(
    numbers, 
    LazyLua.Sort.DESC
)
print(LazyLua.Sort.applyIndices(
    desc_indices, 
    numbers
)) 
-- Output sorted array: 8, 5, 2, 1

-- Bubble sort (stable sort)
local stable_indices = LazyLua.Sort.bubble(
    numbers, 
    LazyLua.Sort.ASC
)

API Reference

Comprehensive documentation of LazyLua's functionality

LazyLua.Math

  • add(a, b) High-precision addition (supports negatives)
  • sub(a, b) High-precision subtraction (supports negatives)
  • mul(a, b) High-precision multiplication (supports negatives)
  • div(a, b, up_to) Division with precision control (default 50)
  • greaterThan(a, b) Check if a > b (supports negatives)
  • lessThan(a, b) Check if a < b (supports negatives)
  • lessThanOrEqual(a, b) Check if a ‚â§ b (supports negatives)
  • greaterThanOrEqual(a, b) Check if a ‚â• b (supports negatives)
  • normalize(s) Normalize number string

LazyLua.String

  • len(str) Get character length (UTF-8)
  • at(str, index) Get character at index (1-based)
  • charat(str, index) Get character at index (0-based)
  • substr(s, start, end_) Get substring (UTF-8 support)
  • split(str, delimiter) Split string by delimiter
  • join(list, separator) Join array with separator
  • replace(s, old, new, limit) Replace substrings (with limit)
  • includes(s, substr) Check if contains substring
  • indexof(s, substr, start) Find first occurrence
  • lastindexof(s, substr) Find last occurrence
  • left(str, n) Get leftmost n characters
  • Repeat(str, n) Repeat string n times

LazyLua.Sort

  • bubble(arr, method) Bubble sort (stable, returns indices)
  • quick(arr, method) Quick sort (faster, returns indices)
  • applyIndices(i, arr) Apply indices to reorder array
  • Sort.ASC Ascending order constant
  • Sort.DESC Descending order constant

Additional Utilities

  • to2DTable(text, rowSep, colSep) Convert text to 2D table
  • column(table2D, index, default) Extract column from 2D table

Performance Notes

Optimized for both small and large data processing

Math Optimization

  • Addition: Chunked processing for numbers > 12 digits (base 10^12)
  • Subtraction: Chunked processing for numbers > 12 digits
  • Multiplication: Naive algorithm (‚â§100 digits), Karatsuba (>100 digits)
  • Division: Long division with configurable precision
  • Negative Numbers: Full support with optimized sign handling

Performance Characteristics

  • Small Numbers: Direct computation for maximum speed
  • Large Numbers: Algorithm switching based on input size
  • String Processing: UTF-8 optimized with minimal overhead
  • Sorting: QuickSort for large arrays, BubbleSort for stability
  • Memory Usage: Efficient string handling to minimize overhead
